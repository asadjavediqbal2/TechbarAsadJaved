POST POST is the HTTP method that is designed to send loads of data to a server from a specified resource. It usually transmits relatively small loads of data to a receiver. This method allows data to be sent as a package in a separate communication with the processing script. This means that data sent through the POST method will not be visible in the URL, as parameters are not sent along with the URL. The format of an HTTP POST should have HTTP headers, followed by a blank line, followed by the request body. POST requests can be used to submit a web form or upload a file, but it is critical to ensure that the receiving application resonates with the format being used. The Content-Type header indicates the type of body in the POST request. For example, in our case content type describes content as JSON and X-Auth header indicates the authentication via API Key. d.
